## Intro

## Prerequisites

- PostgreSQL
- Go 1.24+

## Installation

`go install gator`

## Config setup

Create `.gatorconfig.json` at your home directory.

### Config structure
```json
{
  "db_url": "postgres://YOUR_USE_NAME:@localhost:5432/gator?sslmode=disable",
  "current_user_name": "EXAMPLE_USER_NAME"
}
```
